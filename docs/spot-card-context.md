# Flux Spot Card Context

Flux Spot cards can copy two clipboard shapes for troubleshooting and reproduction.

This page is intentionally linked from each full LLM card export so pasted context has a stable explanation of what the payload means.

## First Click: Card Data

The first click copies a compact Markdown table for quick notes, chat messages, and operator handoff.

The table contains rendered point labels, current values, units, quality, stale state, and canonical tag address for each point in the card.

## Second Click: LLM Export

After the first click, click the same card chip again to copy the full LLM export. This includes card identity, address space, current values, and reproducible JSON.

Treat the address space as the stable definition. Treat current values as transient diagnostic evidence.
